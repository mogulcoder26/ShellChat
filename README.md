<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the Repo and Navigate to chat-app</p>

<p>2. Change HOST number to</p>

```
127.0.0.1
```

<p>3. Start Server Using</p>

```
node server
```

<p>4. Connect with multiple Clients using</p>

```
node client
```
